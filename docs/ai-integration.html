AI Integration

Photo Tales Gallery integrates OpenAI's GPT-4o-mini model for intelligent photo analysis, conversational interviews, and automated content generation.

OpenAI Integration Overview

Conversational AI

Interactive interviews that help users remember and document photo stories through natural conversation.

Content Generation

Automatic generation of summaries, tags, and descriptions based on photo content and context.

Image Analysis

Visual analysis of photos to extract content, objects, scenes, and generate relevant metadata.

AI Models & Configuration

Primary Model: GPT-4o-mini

Model Specifications

  • • Model: gpt-4o-mini
  • • Max Tokens: 300 (conversational)
  • • Temperature: 0.7 (creative but focused)
  • • Cost: ~$0.00015 per 1K tokens

Capabilities

  • • Text generation and conversation
  • • Image analysis (base64 encoded)
  • • Context-aware responses
  • • Multi-turn conversations

AI Prompts & System Design

Prompt Architecture

All AI prompts are centralized in backend/prompts.py for easy maintenance and consistency.

Memory Gatherer Prompt

You are "Memory Gatherer," an interviewer specializing in finding details and facts about a person. Your goal is to help users remember and document the story behind their images through engaging, specific questions that build upon previous context.

Summary Generation Prompt

Write a summary for this image based ONLY on the context information provided. Use exactly these labels: "Title:", "Summary:" and base your summary ONLY on the context information provided.

Context Enhancement

When existing context is available, the system enhances prompts to build upon previous information rather than asking basic questions, creating more engaging and personalized conversations.

API Integration Details

OpenAI API Calls

1. Conversational Interview

response = openai.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": system_prompt},
        {"role": "user", "content": user_input}
    ],
    max_tokens=300,
    temperature=0.7
)

2. Image Analysis

response = openai.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {
            "role": "user",
            "content": [
                {"type": "text", "text": "Analyze this image and generate tags"},
                {"type": "image_url", "image_url": {"url": f"data:image/jpeg;base64,{encoded_image}"}}
            ]
        }
    ],
    max_tokens=150
)

3. Summary Generation

response = openai.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": summary_prompt},
        {"role": "user", "content": context_text}
    ],
    max_tokens=200,
    temperature=0.5
)
