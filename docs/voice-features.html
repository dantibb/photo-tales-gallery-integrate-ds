Voice Features

Photo Tales Gallery includes comprehensive voice interaction capabilities using the Web Speech API for both speech recognition and text-to-speech.

Voice Features Overview

Speech Recognition

Convert your spoken words into text for natural interaction with the AI interviewer.

Text-to-Speech

Hear AI responses read aloud automatically or on-demand for a more engaging experience.

Voice Controls

Toggle voice features, adjust speech settings, and control playback with intuitive controls.

Speech Recognition

How It Works

Speech recognition uses the Web Speech API to convert your spoken words into text in real-time.

Key Features

  • • Real-time speech-to-text conversion
  • • Continuous listening mode
  • • Automatic language detection (en-US)
  • • Visual feedback during recognition
  • • Error handling and fallback options

Usage Instructions

1

Click the microphone button next to the text input field

2

Speak clearly into your microphone

3

Review the recognized text that appears in the input field

4

Click "Send" or press Enter to submit your response

Text-to-Speech

AI Response Reading

Text-to-speech automatically reads AI responses aloud, making conversations more natural and engaging.

Features

  • • Automatic speech synthesis for AI responses
  • • Manual playback control for any message
  • • Configurable speech rate and voice
  • • Visual indicators during speech
  • • Stop/play controls for active speech

Controls

Auto-Speak Toggle
  • • Enable/disable automatic speech
  • • Checkbox in chat interface
  • • Remembers your preference
Manual Playback
  • • Click speaker icon on any message
  • • Play/pause active speech
  • • Individual message control

Best Practices

Speech Recognition Tips

  • • Clear Speech: Speak clearly and at a normal pace
  • • Quiet Environment: Minimize background noise
  • • Microphone Quality: Use a good quality microphone
  • • Browser Permissions: Allow microphone access when prompted
  • • Language: Currently supports English (en-US)

Text-to-Speech Tips

  • • Auto-Speak: Enable for hands-free interaction
  • • Manual Control: Click speaker icons for specific messages
  • • Speech Rate: Adjust for comfortable listening
  • • Voice Selection: Choose preferred voice from available options
  • • Stop Speaking: Click speaker icon again to stop

Privacy & Security

Privacy Features

  • • Voice processing happens locally in the browser
  • • No voice data is sent to external servers
  • • Microphone access is optional and can be revoked
  • • All voice features can be disabled

Security Considerations

  • • HTTPS required for production use
  • • Browser permission management
  • • Secure microphone access
  • • No voice data storage

Troubleshooting

Microphone Not Working

Check browser permissions, ensure microphone is connected and working, and try refreshing the page.

Speech Recognition Issues

Ensure you're using a supported browser (Chrome recommended), speak clearly, and check for background noise.

Text-to-Speech Not Working

Check if auto-speak is enabled, try clicking speaker icons manually, and ensure system audio is working.

Browser Compatibility

Voice features work best in Chrome, Edge, or Safari. Firefox has limited support for speech recognition.
