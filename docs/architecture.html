System Architecture

Photo Tales Gallery follows a modern client-server architecture with cloud-native components and AI integration.

High-Level Architecture

System Overview

Frontend Layer

  • • React 18 + TypeScript
  • • Tailwind CSS + Radix UI
  • • Web Speech API
  • • React Query

Backend Layer

  • • Python Flask
  • • OpenAI Integration
  • • Image Processing
  • • API Gateway

Cloud Layer

  • • Google Cloud Storage
  • • Google Cloud Firestore
  • • OpenAI API
  • • Authentication

Frontend Components

Data Flow Architecture

Request Flow

1

User Interaction: User interacts with frontend (photo selection, voice input, etc.)

2

API Request: Frontend makes HTTP request to Flask backend

3

Backend Processing: Flask routes request to appropriate handler

4

External Services: Backend calls OpenAI API, Google Cloud services

5

Data Storage: Results stored in Firestore database

6

Response: Backend returns JSON response to frontend

7

UI Update: Frontend updates UI with new data
