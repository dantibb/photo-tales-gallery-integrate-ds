Photo Tales Gallery Documentation

Photo Tales Gallery is a modern photo gallery application with AI-powered interview capabilities and voice interaction features.

What is Photo Tales Gallery?

Photo Tales Gallery helps users document and remember the stories behind their photos through conversational AI interactions. It combines modern web technologies with advanced AI capabilities to create an engaging photo management experience.

Voice Interaction

Speak to your photos with voice recognition and text-to-speech capabilities.

AI Interviews

Interactive AI conversations that help you remember and document photo stories.

Cloud Storage

Secure cloud storage with Google Cloud Platform integration.

Key Features

  • AI-Powered Photo Analysis: Automatic tagging, summarization, and metadata extraction
  • Conversational Interface: Natural language interactions with your photo collection
  • Voice Input & Output: Speak to your photos and hear AI responses
  • Context-Aware AI: Remembers previous conversations and builds on them
  • Dual Interface Modes: Choose between avatar and text-based interactions
  • Advanced Search: Find photos by tags, dates, locations, and content
  • Cloud Integration: Seamless Google Cloud Platform integration

Technology Stack

Frontend

  • React 18 with TypeScript
  • Vite build system
  • Tailwind CSS + Radix UI
  • React Router DOM
  • Web Speech API

Backend

  • Python Flask
  • Google Cloud Firestore
  • Google Cloud Storage
  • OpenAI GPT-4o-mini
  • Pillow (PIL)

Quick Start

Terminal 
# Clone the repository
git clone https://github.com/your-username/photo-tales-gallery.git
cd photo-tales-gallery

# Start everything with one command
./start_all.sh

Note: You'll need OpenAI API keys and Google Cloud Platform credentials to use all features. The start_all.sh script will warn you about missing configuration. See the Installation Guide for detailed setup instructions.
